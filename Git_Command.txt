====================================pytest================================================

Rn the code with verbosity
C:\Python34\Scripts\py.test.exe -v tes_matlib.py


Run specific set of test
C:\Python34\Scripts\py.test.exe -v -k sum tes_matlib.py

Run with marker

@pytest.mark.name
C:\Python34\Scripts\py.test.exe -v -m "marker_name" tes_matlib.py

stop test after n failures

pytest --maxfail = <num>


output the reports in xml
pytest --junitxml="result.xml"
